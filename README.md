LCC CPSC-101 Assignment 8


Instructions:

You will perform the following steps:

1.    Install Visual Studio Code locally.
2.   Install sufficient extensions to run Python either directly or as a notebook (Jupyter).
3.   Download the GitHub Desktop client.
4.   Set up a GitHub account.
5.    Write three Python functions that will work together. 
    1.    main() - This function starts the program, requesting either a decimal or hexadecimal input.
    2.    parse_input() - This function determines if the string input represents an integer or a hex (hex begin with "0x").
    3.    convert_to_binary(type) - This function converts the input to binary, taking into account whether the type is decimal or hexadecimal.
6.    Students MUST have at least 15 commits that show iteration as one builds up the functions.
